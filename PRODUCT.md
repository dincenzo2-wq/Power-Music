# Product

## Register

product

## Users
Video editors (professional or hobbyist) using software like DaVinci Resolve or Premiere Pro. They work in fast-paced environments and need extremely quick access to categorized sound assets.

## Product Purpose
A highly efficient, drag-and-drop enabled Power Music organizer. It acts as an external "Power Bin", bypassing the clunky native OS file explorers to provide a tailored, seamless asset-gathering workflow.

## Brand Personality
Professional, Deep, and High-Speed. It should feel like a native module of a high-end NLE (Non-Linear Editor) like DaVinci Resolve.

## Anti-references
- Clunky native OS file explorers (Windows Explorer default views).
- "Gamer" aesthetic (RGB, excessive borders).
- Light-mode web apps.

## Design Principles
- Speed over everything: interactions should be instant, layout should prioritize density without clutter.
- Native feel: it should blend into the dark-mode ecosystem of professional video editing suites.
- Tactile feedback: hovering, dragging, and dropping must have distinct, physical-feeling visual states.

## Accessibility & Inclusion
- High contrast text against dark backgrounds.
- Clear visual indicators for drag-and-drop states.
